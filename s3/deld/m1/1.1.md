# MODULE I: Topic 1.1 - Number System Conversions

**Key Skill:** Converting between Decimal, Binary, and Hexadecimal.

---
### **1. Decimal to Binary**
*   **Method:** "Double Dabble" (Divide by 2).
*   **Procedure:**
    1.  Divide the decimal number by 2.
    2.  Write down the remainder (which will be 0 or 1).
    3.  Divide the result of the division by 2 again. Record the new remainder.
    4.  Repeat until the result is 0.
    5.  Read the list of remainders from the **BOTTOM to the TOP**.

*   **Example: Convert (26)₁₀ to Binary.**
    *   26 / 2 = 13 Remainder **0**
    *   13 / 2 = 6  Remainder **1**
    *   6 / 2  = 3  Remainder **0**
    *   3 / 2  = 1  Remainder **1**
    *   1 / 2  = 0  Remainder **1**
    *   Reading bottom-up: `11010`.
    *   **Answer: (11010)₂**

### **2. Binary to Decimal**
*   **Method:** Sum of Powers of 2.
*   **Procedure:**
    1.  Write the powers of 2 (1, 2, 4, 8, 16, ...) below each binary digit, from right to left.
    2.  Add up the powers of 2 that have a `1` above them.

*   **Example: Convert (11010)₂ to Decimal.**
    *   Binary:  `1   1   0   1   0`
    *   Powers: `16  8   4   2   1`
    *   Sum: `16 + 8 + 0 + 2 + 0 = 26`
    *   **Answer: (26)₁₀**

### **3. Binary to Hexadecimal (Shortcut)**
*   **Method:** Grouping by Fours.
*   **Procedure:**
    1.  Starting from the right, group the binary digits into sets of 4.
    2.  If the leftmost group has fewer than 4 bits, add leading zeros.
    3.  Convert each 4-bit group to its single hexadecimal digit equivalent. (Remember: 10=A, 11=B, ..., 15=F).

*   **Example: Convert (11010110)₂ to Hex.**
    *   Group: `1101` `0110`
    *   Convert `1101`: `8+4+1 = 13`, which is **D**.
    *   Convert `0110`: `4+2 = 6`, which is **6**.
    *   **Answer: (D6)₁₆**

### **4. Hexadecimal to Binary (Shortcut)**
*   **Method:** Expanding to Fours.
*   **Procedure:** Convert each hexadecimal digit into its 4-bit binary equivalent.

*   **Example: Convert (D6)₁₆ to Binary.**
    *   **D** (which is 13) = `1101`
    *   **6** = `0110`
    *   Combine them: `11010110`.
    *   **Answer: (11010110)₂**
